1. Есть основной модуль "APIServices", для работы с нашими "services".

2. В него мы ипортируем методы работы с 'data'.

3. В нем же мы создаем Class "ApiInterection", в котором мы вызываем методы для работы с 'data'.
	- методы создаем статические, т.к. их нужно будет вызывать у самого Class, не создавая каждый раз новые объекты.

4. Создаем 'createCard.js'. В него импортируем 'ApiInterection', для дальнейшего вызова в нем метода 'createNewCard'.
	- получаем домЭлементы 'input' со страницы, для дальнейшего использования их значения.
	- создаем объект 'objForCreate' для хранения данных полученных из 'input.value'.
	- вызываем метод класса ApiInterection.createNewCard(objForCreate) и передаем, как аргумент, наш полученный объект.
(метод 'createNewCard' представляет собой POST запрос, т.о. мы записываем полученный объект в нашу базу).

5. Создаем 'getAllCard'. В него импортируем 'ApiInterection', для дальнейшего вызова в нем метода 'getCard'.
	- создаем функцию и вызываем в ней метод 'ApiInterection.getCard()'.
(метод 'getCard' представляет собой GET запрос, т.о. мы получаем из базы наши объекты)

6. В основной index.js импортируем функцию getAllCard, и вызываем ее.
(т.о. получем данные из нашей базы)
